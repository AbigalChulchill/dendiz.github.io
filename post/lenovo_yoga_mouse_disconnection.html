Lenovo Yoga 2 Pro Bluetooth mouse disconnection issue

2015-03-30

I just got a Microsoft Comfort Sculpt Bluetooth mouse, to free up the USB ports on my Lenovo Yoga 2 pro laptop, but I soon faced the problem that many other users were facing

mouse freezes suddenly mouse disconnects Doing some research quickly revealed quite a bit of people were facing the same issues with this product. The problem is with the windows 8.1 bluetooth drivers. The solution is simple uninstall the current drivers.

Important! Before you proceed make sure your D drive contains the lenovo provided restore utilities including the blue tooth drivers. If you do not have these or are unsure do not proceed!

Press CTRL + X and select device manager from the popup on the bottom right.

image lost

right click on the highlighted item and click uninstall. Select delete drivers check box and remove the driver. Now go to d:\Drivers\Bluetooth\Win7\vs64\ and right click the Setup.exe file. Choose properties and select the compatibility tab. Select windows 7 compatibility mode as shown in the screen shot.

image lost

Now run the setup.exe program and install the driver. The windows 7 drivers will work correctly and the disconnection issues will disappear.

Now open the device manager again and right click on Bluetooth driver in the first image and select properties to open the following dialog

and uncheck the power save option. This will stop the mouse from freezing and will make it responsive.
